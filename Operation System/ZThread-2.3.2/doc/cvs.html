An anonymous CVS can be accessed with the following commands: 
 cvs -d:pserver:anonymous@cvs.ZThread.sourceforge.net:/cvsroot/zthread login <blank password>
 cvs -z3 -d:pserver:anonymous@cvs.ZThread.sourceforge.net:/cvsroot/zthread co -P zthread
Web-based access to the CVS is also available.

It should be noted that sourceforge has changed anonymous CVS access so that it runs of a backup server. This means anything you checkout may be up to 24 hours old. If you need something newer, please contact me directly.

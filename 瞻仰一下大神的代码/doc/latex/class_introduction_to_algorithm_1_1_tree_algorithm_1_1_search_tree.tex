\begin{DoxyItemize}
\item 把{\ttfamily next\+\_\+node}的右子节点剪切到{\ttfamily next\+\_\+node}的位置
\item 将{\ttfamily next\+\_\+node}剪切到{\ttfamily node}的右子位置

Detailed Description

template<typename NodeType>
struct IntroductionToAlgorithm::GraphAlgorithm::List< NodeType >

List：链表数据结构

链表包含两个数据成员：

  • head：指向链表头部的元素的强指针
  • current:指向链表当前处理元素的强指针

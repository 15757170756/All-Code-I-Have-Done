Detailed Description

template<typename KType>
struct IntroductionToAlgorithm::GraphAlgorithm::FrontFlowVertex< KType >

FrontFlowVertex：relabel_to_front算法的图的结点的数据结构，算法导论26章26.4节

FrontFlowVertex 继承自 FlowVertex，它比FlowVertex顶点多了一个N_List数据成员，表示邻接链表

relabel_to_front 算法中，每一个FrontFlowVertex顶点位于两个级别的链表中：

  • L 链表：最顶层的链表，L包含了所有的非源、非汇顶点
  • u.N 链表：某个顶点u的邻接链表

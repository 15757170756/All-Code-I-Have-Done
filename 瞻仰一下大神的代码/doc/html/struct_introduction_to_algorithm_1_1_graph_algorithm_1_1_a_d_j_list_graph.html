Detailed Description

template<unsigned N>
struct IntroductionToAlgorithm::GraphAlgorithm::ADJListGraph< N >

ADJListGraph：图的邻接表表示，算法导论22章22.1节

图的邻接表主要包含一个数据：

  • array：邻接表，类型为std::array<std::vector<std::pair<VIDType,EWeightType>>,N>，为N行，每一行代表一个节点：

为了便于计算，这里并不管理边和顶点，只是维护邻接表。边、顶点与邻接表的同步由使用者确保。

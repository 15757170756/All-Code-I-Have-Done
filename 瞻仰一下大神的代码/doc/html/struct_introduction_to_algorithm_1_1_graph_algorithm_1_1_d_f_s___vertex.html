template<typename KType>
struct IntroductionToAlgorithm::GraphAlgorithm::DFS_Vertex< KType >

DFS_Vertex：用于深度优先搜索的顶点类型，算法导论22章22.3节

深度优先搜索的顶点类型继承自Vertex模板类型，它比Vertex多了四个数据成员：

  • color：顶点的颜色。顶点可以有三种颜色，分别为DFS_Vertex::COLOR::BLACK、DFS_Vertex::COLOR::WHITE、DFS_Vertex::COLOR::GRAY
  • parent: 指向父顶点的强引用
  • discover_time：顶点的发现时间
  • finish_time：顶点的完成时间

广度优先搜索的顶点类型还支持以下两个操作：

  • set_disovered(discover_t)：发现本顶点
  • set_finished(finish_t)：本顶点搜索完毕

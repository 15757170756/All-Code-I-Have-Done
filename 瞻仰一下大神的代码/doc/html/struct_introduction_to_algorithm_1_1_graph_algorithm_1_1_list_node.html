template<typename ValueType>
struct IntroductionToAlgorithm::GraphAlgorithm::ListNode< ValueType >

ListNode：链表结点的数据结构

链表结点包含两个数据成员：

  • value：链表结点保存数据的弱指针（因为该保存数据有个强指针指向本结点）
  • next:指向本链表中当前结点的下一个结点的强指针

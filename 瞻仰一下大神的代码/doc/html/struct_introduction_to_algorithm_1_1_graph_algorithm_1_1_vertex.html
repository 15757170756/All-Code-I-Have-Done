Detailed Description

template<typename KType>
struct IntroductionToAlgorithm::GraphAlgorithm::Vertex< KType >

Vertex：图的顶点，算法导论22章22.1节

顶点是一个结构体的模板，模板类型为它存储的数据的类型。它主要包含两个数据：

  • key:顶点存放的数据
  • id:顶点的编号（从0开始，负编号的节点为无效节点）。它是一个const int，一旦顶点初始化完毕就不可更改。

默认情况下：id为-1,key为T()。
